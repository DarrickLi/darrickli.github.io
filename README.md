# darrickli.github.io

Academic-focused bilingual GitHub Pages site for Darrick Li.

## Structure

- `index.html` - single-page bilingual site (EN/ZH)
- `assets/cv.pdf` - your CV PDF (replace this)
- `content/` - optional markdown drafts for future extension

## Publish (GitHub Pages)

1. Create repo: `DarrickLi/darrickli.github.io`
2. Push `main` branch
3. In repo settings, enable GitHub Pages from `main` / root

Then site URL is:
`https://darrickli.github.io`
